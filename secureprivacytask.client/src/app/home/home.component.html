Marco Olivieri's Test Project for Secure Privacy

In this project, we will present the required points divided into two tasks. Each functionality will be accessible from the respective menu item (Users -> Task 1 | String Analysis -> Task 2).

Task 1: Basic .NET/C# Project with Angular Interface and GDPR Considerations

Objective:

Develop a small .NET/C# web application with a basic Angular front-end, integrating MongoDB, and considering GDPR compliance aspects.

Requirements:

  • Data Structure: Define a simple entity such as User or Product with basic attributes.
  • CRUD Functionality: Implement Create and Read operations.
  • Angular Front-End: Develop a minimal interface for adding (Create) and displaying (Read) the entities, demonstrate awareness of Redux approach.
  • MongoDB Integration: Use MongoDB for data storage and retrieval, and demonstrate using compound indexes in MongoDB.
  • GDPR Considerations: Implement basic GDPR compliance measures at the code and user-interface level.

Task 2: Binary String Analysis Function

Objective:

Write a C# function to evaluate binary strings based on specific criteria.

Requirements:

  • The function accepts a binary string as input.
  • Check if the binary string is 'good' based on these conditions:
    • Equal number of 0's and 1's.
    • For every prefix, the number of 1's is not less than the number of 0's.
